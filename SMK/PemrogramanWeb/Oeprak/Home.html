Membuat Website Sederhana dengan HTML dan CSS

Pada Website yang saya buat, ada beberapa halaman yang tersedia.
Halaman tersebut antara lain, halaman Login, Home, Konten, About, dan juga Contact.

  1. Halaman Login
    Menjelaskan / Berisi Tentang halaman pertama dimana User harus mengidentifikasi dirinya terlebih dahulu agar bisa memasuki halaman berikutnya ( Home )
  2. Halaman Home
    Menjelaskan / Berisi Tentang penjelasan singkat mengenai isi dari Website yang dibuat, dan juga menjadi halaman untuk penjelasan mengenai setiap halaman yang berada pada website
  3. Halaman Konten
    Menjelaskan / Berisi Tentang isi / konten dari halaman Website yang dibuat. Pada kali ini saya mengambil tema Kuliner yaitu Sate. Akan ada penjelasan singkat mengenai kuliner Sate yang ada di Indonesia.
  4. Halaman About
    Menjelaskan / Berisi Tentang data diri penulis / pembuat website ini, contohnya tempat tanggal lahir, tempat tinggal, hobi, dan lain-lain.
  5. Halaman Contact
    Menjelaskan / Berisi Tentang sarana untuk menghubungi penulis / pembuat website. Adapun caranya yaitu dengan mengisi nama, alamat Email, serta pesan yang ingin disampaikan kepada penulis.
